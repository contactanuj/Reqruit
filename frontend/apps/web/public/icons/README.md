# PWA Icons

The following icon files are required for the PWA manifest (`public/manifest.json`):

- `icon-192.webp` — 192x192 app icon (used for home screen on most devices)
- `icon-512.webp` — 512x512 app icon (used for splash screen and app stores)

Generate these from the Reqruit logo/brand asset. Both sizes should have transparent
backgrounds for the `any` purpose variant; the `maskable` variant should include
safe-zone padding (at least 10% on each side).
